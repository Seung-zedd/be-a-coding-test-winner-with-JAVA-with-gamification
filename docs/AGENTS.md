# ⚔️ 코딩 테스트 정복 가이드 (ConquerTheCode Guide) ⚔️

이 문서는 용사(USER)와 부관(AI)이 협력하여 코딩 테스트라는 던전을 공략하기 위한 **전투 교본**입니다. 우리는 아래와 같은 특수 주석 시스템을 사용하여 효율적으로 소통합니다.

---

### 🛡️ 주석 시스템 (Commenting System)

| 기호        | 역할              | 설명                                                                                         |
| :---------- | :---------------- | :------------------------------------------------------------------------------------------- |
| `/* ... */` | **재도전 퀘스트** | 클래스 선언 **외부(상단)**에 부관(AI)이 배치하는 챌린지 힌트입니다. 제출 시 지우기 편합니다. |
| `//`        | **로직 설계**     | 용사님이 코드를 작성하기 전, 생각의 흐름을 정리하는 의사코드입니다. (메서드 내부)            |
| `//?`       | **복습 포인트**   | 문제 풀이 후 의문점이나 다시 볼 때 필요한 팁을 기록합니다. (클래스 내부 또는 외부)           |
| `//!`       | **경고 및 실수**  | 저질렀던 실수나 필수 주의 함정을 표시합니다. (메서드 내부)                                   |

---

### 🏆 던전 공략 랭킹 시스템 (Ranking System)

전투 종료 후, 용사님의 활약에 따라 다음과 같이 랭크를 부여합니다.

| 랭크  | 명칭        | 기준                                                                              |
| :---: | :---------- | :-------------------------------------------------------------------------------- |
| **S** | **Legend**  | 에이전트의 도움 없이 스스로 완벽하게 문제를 해결했을 때                           |
| **A** | **Expert**  | 로직 구현 중 막힘이 있어 에이전트의 도움(디버깅, 코드 수정 등)을 받아 해결했을 때 |
| **B** | **Knight**  | 알고리즘 로직 설계 자체를 못해서 에이전트의 가이드를 전적으로 받았을 때           |
| **C** | **Trainee** | 문제 지문 자체를 이해하지 못해 에이전트의 상세 설명이 필요했을 때                 |
